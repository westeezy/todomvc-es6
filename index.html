What is this? Demo Application

This is a demonstration of using ES6 features today across browesers without being dependent on any client side libraries.

The popular idea from was used to showcase a handful of ES6 features.

Which features?

  • Modules
  • Classes
  • Fat Arrow
  • Generators
  • Template Strings

How to build it?

Built with Grunt, Browserify, and 6to5.
Tested with Mocha and Sinon.
And a few other plugins to tie it all together.
